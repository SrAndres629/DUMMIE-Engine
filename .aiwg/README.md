# AIWG (Artificial Intelligence Working Group) Memory

Este directorio contiene la memoria de trabajo, protocolos, contexto global y estado de sesiones del motor DUMMIE.

Por política de versionado (ver `control/AIWG_VERSIONING_POLICY.md`), la memoria de trabajo generada (eventos, logs de sesión) NO se versiona en el repositorio principal, mientras que los protocolos de control y configuraciones sí.
